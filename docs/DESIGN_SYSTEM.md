# Design System

## Base UI Components (`src/components/ui/`)

### Button (`Button.tsx`)
- **Variants**: default, outline, ghost, accent.
- **Sizes**: sm, md, lg.
- **Features**: Disabled states, loading states, icon support.

### Input (`Input.tsx`)
- Standard text input with focus rings and error states.

### Checkbox (`Checkbox.tsx`)
- Custom styled checkbox.

### Modal (`Modal.tsx`)
- Backdrop blur, animated entrance (Framer Motion).

### Logo (`Logo.tsx`)
- Brand identity component.
