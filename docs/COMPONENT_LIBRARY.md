# Component Library

## Pages
- Login, Signup, ForgotPassword, ResetPassword
- Dashboard (Freelancer)
- ProfileManager (Freelancer)
- ClientDashboard
- CompanyPage
- ClientJobsPage
- CreateEditJobPage
- FreelancerJobsPage

## Organisms
- **ExperienceTimeline**: Displays work history.
- **EducationCards**: Displays academic history.
- **PortfolioGrid**: Masonry/Grid of projects.

## Molecules
- **CompanyCard**: Summary of client's company.
- **JobCard**: Summary of a job posting.
- **ProfileProgressCard**: Gamified profile completion.
- **SkillChips**: List of skills with delete action.

## Atoms
- Button, Input, Checkbox, Logo.
