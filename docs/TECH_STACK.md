# Tech Stack

## Frontend
- React 19
- Vite
- Tailwind CSS v4
- Framer Motion (Animations)
- React Router v7
- TanStack Query (Data Fetching)
- React Hook Form + Zod (Validation)
- Lucide React (Icons)
- Axios

## Backend
- Node.js
- Fastify
- Prisma ORM
- PostgreSQL
- @fastify/jwt (Auth)
- @fastify/cookie
- @fastify/oauth2
- @fastify/multipart (File Uploads)
- bcryptjs (Hashing)
- Zod (Validation)

## Deployment
- Frontend: Vercel
- Backend: Render
- Database: Neon / Render Postgres
